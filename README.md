# LearningPlayWright

Playwright learning exercises and example scripts.

Project structure

- Chapter_01_Basics/: basic Playwright setup and simple scripts
- Chapter_02_JS_Basics/: JavaScript fundamentals used in examples

Quick start

1. Install dependencies (if you add a package.json):

```powershell
npm install
```

2. Run Playwright tests (if configured):

```powershell
npx playwright test
```

Notes

- This repo contains example JS files and a `.gitignore`.
- Add a `package.json` and Playwright config to run tests.
